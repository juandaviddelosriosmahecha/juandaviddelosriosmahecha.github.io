  • 2025 - 2025

    Github

    Gray scott reaction diffusion hpc
    • Gray-Scott Reaction–Diffusion HPC Simulator is a computational simulation project developed in C++ that implements reaction–diffusion models based on the Gray–Scott equations. The project is designed to model complex, self-organizing patterns that emerge in dynamical systems, using high-performance computing (HPC) techniques to optimize execution. The implementation includes CUDA acceleration, enabling the use of GPU computing power to efficiently simulate large-scale systems. The codebase is modularly structured and includes Bash automation scripts for running experiments and organizing simulation results.
  • 2025 - 2025

    Github

    Ising hpc phase transition
    • I developed a computational project focused on the simulation of the Ising Model using high-performance computing (HPC) to investigate phase transitions. This repository contains the implementation of optimized algorithms to simulate magnetic behavior at the computational level, applying parallelization techniques and distributed processing to analyze critical phenomena in physical systems.
  • 2025 - 2025

    Github

    Neural Network Fitting Damped Oscillator Analysis
    • Development of a quantitative analysis project that integrates neural networks with numerical methods to model and fit the behavior of damped oscillators. The project combines machine learning techniques with differential equations, enabling the training of neural network models to capture the complex dynamics of oscillatory systems under damping.
